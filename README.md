## 外部ライブラリのインストール
`npm i`

## twitterへのtoken設定
index.jsファイル内の下記のソースコードに必要な情報を設定してください。
```
const consumerKey = '';       // CONSUMER_KEY
const consumerSecret = '';    // CONSUMER_SECRET
const accessTokenKey = '';    // ACCESS_TOKE
const accessTokenSecret = ''; // ACCESS_TOKEN_SECRET
```

## 実行コマンド
`node index.js`

## 結果出力ファイル
画像をimg配下に出力します。
